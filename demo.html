This web app is a very simple demo / proof-of-concept for a speech recognition based language learning website. Functionality is very simple (and occasionally buggy!), as it was my first ever foray into Javascript.

A fully fledged version would allow multiple languages, scenarios, track words mispronounced, allow the user to hide words to increase difficulty level, etc...
